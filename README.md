O ambiente básico é Linux com Python 3.6 e Python-Notebook instalado. 

Para instalar o ambiente usando Anaconda os passos são os seguintes:

1. Download e instalação do software Anaconda: [anaconda](https://www.anaconda.com/products/individual)
2. Criação de um ambiente de desenvolvimento Python 3.6 (no terminal Linux).

`conda create -n estruturas_de_dados python=3.6`

3. Ativação do ambiente criado:

`conda activate estruturas_de_dados`

4. Instalação do git, Python-Notebook, graphviz e plotly no ambiente estruturas_de_dados

`conda install -c conda-forge jupyter git matplotlib python-graphviz -y`

5. Clone este repositório

`git clone https://github.com/giiff/estruturas_de_dados_python.git`

7. Rode o servidor Python-Notebook de DENTRO do diretório clonado:

`jupyter-notebook`

Se preferir não instalar o ambiente e usar on-line no [Google Colab](https://colab.research.google.com) basta colar o endereço GitHub: https://github.com/giiff/estruturas_de_dados_python.git
